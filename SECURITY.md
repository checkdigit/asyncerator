# Security Policy

## Supported Versions

These versions of `asyncerator` are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| \>= 2.x | :white_check_mark: |
| \< 2.0  | :x:                |

## Reporting a Vulnerability
